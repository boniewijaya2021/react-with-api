# react-with-api

langkah2 
```bash
mkdir react-app-backend
```
```bash
cd react-app-backend
```
```bash
mkdir server
```
create index.js di folder server
lalu 
```bash
npm i express
```

```bash
npm start
```

```bash
npx create-react-app client
```
